☁️ Enable cloud sync

Sync your tasks across all devices using a free GitHub Gist. Takes about 2 minutes to set up.

1
Go to github.com/settings/tokens/new
2
Give it any name (e.g. "todo sync"), set No expiration, check the gist scope, click Generate token
3
Copy the token (starts with ghp_) and paste it below
